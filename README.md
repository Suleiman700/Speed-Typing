# Speed Typing

A simply JavaScript game to test your typing speed.

This can be a good demo for people who are trying to learn JavaScript :)

---

Features:
* Counts errors while typing.
* Timer.
* Counts words per minute.
* Multiple sentences.
* Responsive.
